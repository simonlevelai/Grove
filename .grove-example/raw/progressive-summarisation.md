# Progressive Summarisation

Progressive Summarisation is a note-processing technique developed by Tiago Forte. It addresses a fundamental tension in note-taking: capturing everything for future reference versus processing everything for immediate understanding. Forte's solution is to defer processing and apply it incrementally across multiple encounters with a note.

## The Five Layers

**Layer 1: Original notes.** The full captured text — an article, a book highlight, a meeting transcript. No processing applied.

**Layer 2: Bold passages.** On the first revisit, bold the passages that stand out as most relevant. This takes seconds and dramatically improves scannability.

**Layer 3: Highlighted passages.** On a subsequent revisit, highlight the most important bolded passages. This creates a two-tier visual hierarchy within the note.

**Layer 4: Executive summary.** Write a brief summary at the top of the note in your own words. This is the first layer that requires original thought rather than selection.

**Layer 5: Remix.** Use the distilled material as raw input for original work — an article, a presentation, a project deliverable.

## Just-in-Time Processing

The critical insight is that layers are applied only when you return to a note, not when you first capture it. Most notes are never revisited, so processing them upfront wastes effort. The notes that are revisited receive increasingly deep treatment exactly when that treatment is needed.

## Relationship to Other Methods

Progressive Summarisation complements both PARA and Zettelkasten. In a PARA system, it provides a lightweight way to process notes stored in Resources. In a Zettelkasten, it offers a path from raw capture to the polished atomic notes the system requires.
