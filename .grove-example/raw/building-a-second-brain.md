# Building a Second Brain

Building a Second Brain (BASB) is a personal knowledge management methodology created by Tiago Forte. The central premise is that human biological memory is unreliable for the volume and complexity of information modern knowledge workers encounter, and that an external "second brain" — a trusted digital system — can compensate.

## The CODE Framework

BASB is structured around four steps:

**Capture.** Save information that resonates with you. Forte emphasises resonance over comprehensiveness: capture what strikes you as interesting, surprising, or useful rather than trying to save everything. The filter is emotional and intuitive, not systematic.

**Organise.** File captured information using the PARA method (Projects, Areas, Resources, Archives). Organisation is by actionability, not by topic. The question is not "what is this about?" but "what is this useful for?"

**Distil.** Find the essence of each note using Progressive Summarisation. The goal is to make notes progressively more discoverable and usable over time without requiring a large upfront investment.

**Express.** Turn your accumulated knowledge into tangible output — articles, presentations, products, decisions. Forte argues that knowledge has no value until it is expressed. The second brain exists to serve creative and professional output, not to be an end in itself.

## Key Ideas

BASB treats personal knowledge management as a creative practice. The system should enable serendipitous connections, reduce the friction of starting new projects, and build confidence that important information is accessible when needed.

Forte also emphasises "intermediate packets" — small, reusable units of work (summaries, outlines, drafts, diagrams) that can be recombined across different projects. This reduces the cost of starting from scratch and makes past effort compound.
