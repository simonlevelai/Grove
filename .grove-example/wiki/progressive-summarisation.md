# Progressive Summarisation

Progressive Summarisation is a note-processing technique developed by Tiago Forte. It addresses a fundamental tension in note-taking: capturing everything for future reference versus processing everything for immediate understanding. Forte's solution is to defer processing and apply it incrementally across multiple encounters with a note [source: progressive-summarisation.md].

## The Five Layers

**Layer 1: Original notes.** The full captured text — an article, a book highlight, a meeting transcript. No processing applied [source: progressive-summarisation.md].

**Layer 2: Bold passages.** On the first revisit, bold the passages that stand out as most relevant. This takes seconds and dramatically improves scannability [source: progressive-summarisation.md].

**Layer 3: Highlighted passages.** On a subsequent revisit, highlight the most important bolded passages. This creates a two-tier visual hierarchy within the note [source: progressive-summarisation.md].

**Layer 4: Executive summary.** Write a brief summary at the top of the note in your own words. This is the first layer that requires original thought rather than selection [source: progressive-summarisation.md].

**Layer 5: Remix.** Use the distilled material as raw input for original work — an article, a presentation, a project deliverable [source: progressive-summarisation.md].

## Just-in-Time Processing

The critical insight is that layers are applied only when you return to a note, not when you first capture it. Most notes are never revisited, so processing them upfront wastes effort. The notes that are revisited receive increasingly deep treatment exactly when that treatment is needed [source: progressive-summarisation.md].

## Role in Building a Second Brain

Within the [[para-and-second-brain]] methodology, Progressive Summarisation serves as the "Distil" step of the CODE framework. It provides a lightweight way to process notes stored in Resources and other PARA categories without requiring the sustained effort of writing full [[evergreen-notes]] [source: building-a-second-brain.md].

## Complementary Approaches

Progressive Summarisation complements both PARA and the [[zettelkasten]]. In a PARA system, it provides a path from raw capture to usable reference material. In a Zettelkasten, it offers a graduated path from raw capture to the polished [[atomic-notes-principle]] the system requires [source: progressive-summarisation.md].
